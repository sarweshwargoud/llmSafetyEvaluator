System Architecture

Hybrid Machine Learning Approach

Our system employs a sophisticated dual-model approach combining supervised and unsupervised learning for comprehensive threat detection.

Supervised Detection
  • TF-IDF Vectorization: N-gram feature extraction (1-3 grams)
  • Random Forest Classifier: 100 estimators with calibrated confidence
  • Known Pattern Detection: Trained on injection and jailbreak samples
Unsupervised Detection
  • Isolation Forest: Anomaly detection for novel threats
  • Outlier Detection: Identifies unusual prompt patterns
  • Zero-day Protection: Catches previously unseen attack vectors
